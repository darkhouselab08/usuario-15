# memory/ — Knowledge base del proyecto

Memoria de largo plazo del orquestador. `CLAUDE.md` (en la raíz) es la memoria *de trabajo* — corta y siempre cargada. Esta carpeta es la memoria *completa*: se consulta cuando hace falta profundidad.

## Archivos

| Archivo | Contenido | Prefijo |
|---------|-----------|---------|
| `decisiones.md` | Decisiones de negocio o arquitectura tomadas | `DEC-` |
| `aprendizajes.md` | Lo que el sistema o Franco aprende y no debe olvidar | `APR-` |
| `glosario.md` | Jerga, acrónimos, nombres propios, atajos de Franco | — |

## Regla

Cuando algo merece recordarse más allá de la sesión, se escribe aquí con su prefijo y fecha (YYMMDD). Si es crítico para cada sesión, además se resume en `CLAUDE.md`.
